Démarrer avec EstimLink

Tout ce qu'il faut pour installer l'add-in dans Excel et créer votre premier devis.

1. Télécharger le manifeste

EstimLink s'installe via un fichier manifeste (EstimLink-manifest.xml) qui indique à Excel où charger l'add-in.

⬇ Télécharger EstimLink-manifest.xml

2. Installer dans Excel sur le web

  1. Ouvrez un classeur sur excel.office.com (ou office.com → Excel).
  2. Onglet Accueil → Compléments → Plus de compléments.
  3. Onglet Mes compléments → Charger mon complément (Upload My Add-in).
  4. Sélectionnez le fichier EstimLink-manifest.xml téléchargé, puis validez.
  5. L'onglet Estim Link apparaît dans le ruban : cliquez sur le bouton pour ouvrir le panneau.

3. Installer dans Excel pour Windows

Sur le poste de travail, Excel charge les compléments personnalisés depuis un catalogue de confiance (dossier partagé) :

  1. Créez un dossier, par exemple C:\EstimLink, et copiez-y le manifeste.
  2. Partagez ce dossier : clic droit → Propriétés → Partage → Partager (avec vous-même), et notez le chemin réseau (ex. \\VOTRE-PC\EstimLink).
  3. Dans Excel : Fichier → Options → Centre de gestion de la confidentialité → Paramètres du Centre… → Catalogues de compléments approuvés.
  4. Collez le chemin réseau dans URL du catalogue, cliquez sur Ajouter le catalogue, cochez Afficher dans le menu, puis OK et redémarrez Excel.
  5. Ensuite : Accueil → Compléments → Plus de compléments → Dossier partagé → sélectionnez Estim Link → Ajouter.

💡 Cette manipulation n'est à faire qu'une seule fois. Aux prochains démarrages, EstimLink sera directement disponible dans vos compléments.

4. Se connecter

  1. Ouvrez le panneau Estim Link depuis le ruban.
  2. Connectez-vous avec Google ou créez un compte avec votre email.
  3. Vos bibliothèques de prix et projets sont liés à votre compte et vous suivent partout.

5. Premier devis en 3 commandes

Dans le chat du panneau, essayez par exemple :

  • Crée une table DQE pour une villa R+1 — l'assistant structure les lots et postes.
  • Applique les prix de ma bibliothèque — l'Agent Prix rapproche chaque poste d'un prix unitaire.
  • Génère le récapitulatif et convertis en bordereau — totaux, montants en lettres, mise en forme.

6. Connecter ArchiCAD

Depuis l'onglet ArchiCAD du panneau, chargez les nomenclatures exportées de votre maquette. EstimLink résout ensuite chaque référence en cascade (correspondance exacte → normalisée → similarité IA) pour injecter les quantités du modèle dans vos postes.
