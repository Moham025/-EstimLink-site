Démarrer avec EstimLink

Tout ce qu'il faut pour installer l'add-in dans Excel et créer votre premier devis.

1. Installer dans Excel pour Windows (recommandé)

Téléchargez et lancez EstimLink-Excel-Setup.exe (2 Mo) : il enregistre le complément dans votre Excel automatiquement (catalogue de confiance + manifeste). Une seule confirmation Windows (UAC) vous sera demandée pour créer le partage local du catalogue.

⬇ Télécharger EstimLink-Excel-Setup.exe

  1. Lancez l'installeur et suivez les étapes (quelques secondes).
  2. À la fin, la fenêtre de vérification confirme que tout est prêt.
  3. Dans Excel : Insertion → Mes compléments → Dossier partagé → Estim Link → Ajouter (une seule fois ; ensuite l'onglet reste disponible).

💡 Le raccourci « Vérifier EstimLink pour Excel » du menu Démarrer rouvre la fenêtre de diagnostic à tout moment et répare l'installation si besoin.

2. Installer le connecteur ArchiCAD (optionnel)

Si vous utilisez ArchiCAD, installez ensuite EstimLink-Agent-Setup.exe (45 Mo) : l'agent local qui relie vos maquettes au devis. Il installe aussi automatiquement l'add-on Tapir (open source, licence MIT) dans chaque Archicad 27, 28 ou 29 détecté sur votre poste.

⬇ Télécharger EstimLink-Agent-Setup.exe

  1. Fermez Archicad avant l'installation (sinon la copie de Tapir sera refusée).
  2. Lancez l'installeur ; une confirmation Windows (UAC) est demandée pour copier Tapir.
  3. L'agent démarre automatiquement et se relancera à chaque ouverture de session Windows.

3. Installer dans Excel sur le web (alternative sans installeur)

  1. Téléchargez le manifeste seul (EstimLink-manifest.xml).
  2. Ouvrez un classeur sur excel.office.com (ou office.com → Excel).
  3. Onglet Accueil → Compléments → Plus de compléments.
  4. Onglet Mes compléments → Charger mon complément (Upload My Add-in).
  5. Sélectionnez le fichier téléchargé, puis validez : l'onglet Estim Link apparaît dans le ruban.

4. Se connecter

  1. Ouvrez le panneau Estim Link depuis le ruban.
  2. Connectez-vous avec Google ou créez un compte avec votre email.
  3. Vos bibliothèques de prix et projets sont liés à votre compte et vous suivent partout.

5. Premier devis en 3 commandes

Dans le chat du panneau, essayez par exemple :

  • Crée une table DQE pour une villa R+1 — l'assistant structure les lots et postes.
  • Applique les prix de ma bibliothèque — l'Agent Prix rapproche chaque poste d'un prix unitaire.
  • Génère le récapitulatif et convertis en bordereau — totaux, montants en lettres, mise en forme.

6. Connecter ArchiCAD

Le connecteur ArchiCAD (étape 2) doit être installé, et Archicad ouvert avec votre projet. Depuis l'onglet ArchiCAD du panneau, chargez votre maquette : EstimLink résout chaque référence en cascade (correspondance exacte → normalisée → similarité IA) pour injecter les quantités du modèle dans vos postes.
